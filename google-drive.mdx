## What is Google Drive

[Google Drive](https://drive.google.com/?ref=filloutforms-helparticle) is a cloud storage service that lets users to store and access files from any device with an internet connection. It offers features such as file synchronization, file sharing, and collaborative editing.

## How it works

### 1. Connect your Fillout form to Google Drive

Go to **Integrate**, and select `Google Drive`.

<Frame>
  ![Screenshot of the Fillout form creation page](/images/Screenshot2025-02-14at11.43.06AM.png)
</Frame>

`Sign in with Google` to allow Fillout to link to your account. Select a `Folder` you would like respondents to upload their files to.

<Frame style={{ width: '576px', height: '301px' }}>
  ![Screenshot of the Fillout form creation page](/images/Screenshot2025-02-14at11.52.38AM.png)
</Frame>

### 2. Map your form fields on Fillout

Under **Upload fields**, click `+ Add new` followed by a file uploader field.

<Frame style={{ width: '384px', height: '324px' }}>
  ![Screenshot of the Fillout form creation page](/images/Screenshot2025-02-14at11.54.00AM.png)
</Frame>

### 3. Publish and share

Click `Finish setup` followed by `Publish` on the upper right corner, and you're all set\!

## Add a dynamic file path

You can set a dynamic file path to make a new subfolder within your chosen folder. Turn on the `Custom subfolder and file name?` switch. Enter a name for your **subfolder** or click `+` to map a field in your form.

<Frame style={{ width: '336px', height: '232px' }}>
  ![Screenshot of the Fillout form creation page](/images/Screenshot2025-02-14at11.57.25AM.png)
</Frame>

## Add a filename prefix

You can enter a **Filename prefix** to add specific characters at the beginning of each file name upon form submission. Similar to the steps above, type a name or add a reference by clicking the `+` button.

<Frame style={{ width: '336px', height: '70px' }}>
  ![Screenshot of the Fillout form creation page](/images/Screenshot2025-02-14at11.57.34AM.png)
</Frame>
