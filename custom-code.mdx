## Overview

Adding **custom code** into your Fillout forms can be useful to:

- Inject analytics tracking scripts like Google Tag Manager
- Add HTML widgets like Intercom to the page

If you’re looking to style your form, use the [custom CSS feature](/custom-css) instead.

<Note>
  **Note:** Custom CSS is available on the **Business** and **Enterprise** plans. For more details on pricing, click [**here**](https://www.fillout.com/pricing).
</Note>
## How to add a custom code


<Check>
  **Important:** To run a custom code, you’ll first need to connect a [custom domain](/custom-domains). Custom code will only run on live forms (not in preview mode) on your custom domain.
</Check>

### 1. Go to the Settings tab on your form
Click `Custom code`.

<Frame>
  ![](/images/advanced/image-17.png)
</Frame>

### 2. Type your code
Type your code in the `Custom code` field.

<Frame style={{ width: '432px' , height: '223px' }}>
  ![](/images/advanced/image-18.png)
</Frame>

### 3. Publish your form and try it out!
  Press `Save` afterwards.

<Frame style={{ width: '192px' , height: '333px' }}>
  ![](/images/advanced/image-19.png)
</Frame>
