---
title: Get form metadata
api: GET /forms/{formId}
description: "Given the `formId`, returns all the questions in that form and other metadata."
contentType: application/json
---


## Request parameters:

### URL parameters
<ParamField path="formId" type="string">
  The public ID of your form that can be retrieved from `/v1/api/forms`
</ParamField>
