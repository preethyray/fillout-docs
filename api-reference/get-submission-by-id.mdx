---
title: Get submission by id
api: GET /v1/api/forms/{formId}/submissions/{submissionId}
description: 'Same as the “Get all submissions” endpoint but only returns a single item'
contentType: application/json
---

## Request

### URL parameters

<ParamField path="formId" type="string" required>
  The public identifier of the form for which you want to retrieve submissions. This should be a string. If your form’s URL is https://form.fillout.com/t/gHretr3UKYus, the form ID is gHretr3UKYus
</ParamField>

<ParamField path="submissionId" type="string" required>
  The identifier of the submission
</ParamField>

### Query parameters

<ParamField query="includeEditLink" type="boolean" optional>
  Pass `true` to include a link to edit the submission as `editLink`
</ParamField>
